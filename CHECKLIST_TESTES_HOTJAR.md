# Checklist de Testes com Usuários - Hotjar

## Antes dos testes

- [ ] Projeto publicado online
- [ ] Script do Hotjar inserido no `index.html`
- [ ] Instalação verificada no painel do Hotjar
- [ ] Site acessível por link público
- [ ] Mapa de calor criado/ativo no Hotjar

## Usuário 1

- [ ] Acessou o site
- [ ] Rolou até o final da página
- [ ] Observou o dashboard
- [ ] Clicou em cards/personagens
- [ ] Clicou nos botões de interação
- [ ] Explorou a sidebar
- [ ] Tempo mínimo de navegação: 2 a 3 minutos
